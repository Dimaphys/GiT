First level: 1. Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?

2. Как вернуть файл из индекса в рабочий каталог?
git restore

3. Как отменить ошибочный commit ?
git reset
4. Как создать новую ветку и переключится в нее?
git branch input
git checkout input
or
-b: git checkout -b <name>

5.В чем отличение между fast-forward merge  and non fast-forfard merge?
При non fast-forward создаютли новые сущности в виде сохранения истории комитов, это не всегда нужно.

6. Когда может возникнуть конфликт слияния веток, как  его разрешить?
Если изменения внесенные в одну ветку конфликтуют с изменениями в другой. 
Для разрешить конфликт нужно выполнить команду git status- получаем список файлов с конфликтами слияния.
Потом git log --merge получаем список конфликтующих фиксаций в исходной и целевой ветвях.
Далее в текстовом файле можно посмотреть по разделителям <<<<<<< и ======= целевая версия ветви конфликтующего текста.
======= и >>>>>>> исходная ветка конфлектующего текста и далее можем выбрать что нам нужно.
7. Можно ли удалить ветку? Как? 
Да можно git branch.
